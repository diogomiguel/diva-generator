## Key Style Guidelines for ALL

- **Very simple, flat design** - no realistic shading, no gradients
- **Minimal details** - basic shapes only
- **Solid colors** - use the specified color palettes
- **Clean lines** - simple outlines
- **Minimal features** - basic dots for eyes, simple curves for mouth
- **No texture or patterns** - flat fills only
- **Copeable size** - 1000x1000px PNG for easy use
- **Scalable** - simple designs work at any size
